Parallel List Ranking - Helman & JaJa

March 18, 2021

Recently I implemented a list ranking algorithm “Designing Practical Efficient Algorithms for Symmetric Multiprocessors” by Helman & JaJa, in my OMSCS course Intro to High Performance Computing. You can download the paper here. There’s also a very helpful introduction by a fellow classmate, which makes it much easier to understand the concepts. But if you want to implement the algorithm, don’t skip the paper, as a matter of fact, implement it faithfully or you code may end up in a very bad place.

The most interesting part is to implement the algorithm with OMP and CUDA, both are fantastic tools for parallel computing. Although my implementations still have a few sequential parts, the final score shows a ~13x speedup for OMP and a whopping 20.9x for CUDA. Gustafson’s law rocks!
